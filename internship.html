Game Developer Intern

Company - Cya Live (visit website)

Duration - September, 2021 to December, 2021

Technology - Unity Engine, C#, Custom Websocket API

About the Game - Sketcharoo :

  • A similar version of popular board game "Pictionary".
  • Players will use CYA's platform to communicate for guessing the drawing.
  • Moderators/hosts have control of the game flow (i.e. they will decide who will be the drawer for each round).

My Contribution :

  • Implement the gameplay mechanics of multiplayer board game using custom websocket API in Unity Engine.
  • UI programming and animations (Designers from team created UI of all versions, my task was to add those UI in Unity Engine).
  • Optimize the code structure and workflow of multiplayer game to reduce workload of data handling on server.
  • Iterate, test and update mechanics and UI of the game as per design requirements.

Details about diffrent versions shown in video :

  • Version 0 - Senior from the team has implemented networking part using websocket, and created line sync demo for testing the SDK (Video not available).
  • Version 1 - Implemented different colors, different brush sizes, eraser for the game (similar to Whiteboard Application).
  • Version 2 - Implemented the "Draw and Guess" functionality of the game (only drawer can see the Drawing Toolbar and Word to draw, getting random words, etc). Also added color picker and brush size slider.
  • Version 3 - Implemented more user friendly UI for better understanding of the gameflow.
  • Version 4 - Implemented musicFX and UI animations.
  • Another Game - Implemented Lobby settings sync, target sync and few UI animations.
