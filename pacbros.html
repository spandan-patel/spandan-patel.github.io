PacBros - Pacman Clone with Multiplayer

Game Engine - Unity Engine, C#, Photon Network Framework

Team Members - 5 members

Learning Outcome - Basics of Multiplayer Games

About the Game :

  • Same play style like Pacman, but with upto 4 players.
  • Collect points and watch out for all the Ghosts. Eat cherry to activate ability to eat Ghosts.
  • Join the lobby created by your friend or create one yourself.
  • Play with others and collect the most points to WIN the game.

Responsibility :

  • Collaborated with other team members to create the clone of classical retro PACMAN game with twist of Multiplayer functionality.
  • Implemented Player controller and other Player mechanics.
  • Implemented Enemy AI using Unity Engine's built-in NavMesh system for chasing, avoidance and waypoint navigation behaviours.
  • Was responsible for implementing other Gameplay mechanics.
